Planet Memory Game

How to Play:

Click on a card and then select a second card. If you select a matched pair, the cards will turn blue. If you select an unmatched pair, they will flip back over to yellow.

Once you have found 8 pairs, the game ends. Select the restart button at any time to start again.

Enjoy!
