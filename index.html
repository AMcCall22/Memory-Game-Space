Hello!

Welcome to a planet memory game!

How to Play:

Click on your first card and select a second to pair it with. If you select a matched pair, the cards will turn blue. If you select an unmatched pair, they will flip back over to yellow.

Once you have found 8 pairs, the game ends. Select the restart button at any time to start again.

Enjoy!
